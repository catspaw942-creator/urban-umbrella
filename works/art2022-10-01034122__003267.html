各位好我是子貓玲，

Cyberpunk風格的貓貓女完結灑花～

雖然說是Cyberpunk風格啦，但其實也沒這麼Cyberpunk，前前後後的製作上經過了大量修改，幾乎都整個重畫了，啊～實際上人物的部份確是直接砍掉重畫，重畫後又追加改造人風格，然後還多了個外送設定，最後還要加上眼鏡加點知性氣質。

( 阿哩？～改造人還戴眼鏡 !? 啊是裝到次品的眼睛是唄～ )

機車的部分中途也幾經修改，引擎也打掉重製，車體的部分也重新強調了光影與反射的效果。

( 改個角度就差點翻繪圖板了 )

後面也許還有要陸續修改的部分，不過因該不會再改太多了啦，

( 等等～我先準備一下球棒再談 )

總之還算是順利完成目標嚕。

做為自家的貓娘就辛苦她幫忙多貼補些家用嚕，年底應該能多加兩塊豆腐乳。

這裡也題外的感謝一下各位辛苦的外送哥與姊們，多虧了您們生活越來越方便，日子也更多點愉悅，台灣有你們真好。

下面開始外送貓女殺手的技術總結。

( 借用一下梗嚕～ )

第一，貓女的部份肉質要選用上等的，可以採用性感妖艷的素體。

( 建議同學們多嘗試各種風格與姿勢 )

第二，由於機車真的很機車，剛開始多點耐心處理細節可以省下很多麻煩。

( 2D是不能修改角度的，最好草稿一開始就確認完整度 )

第三，LOGO素材雖然很可愛～但是肉真香！

關注沒人理會的畫家子貓玲，這裡總有一款屬於你的菜。
